How to Add a Widget to Your Site

  1. Choose a widget from the options above.
  2. Copy the embed code by clicking the "Copy" button.
  3. Paste the code into your HTML where you want the calculator to appear.
  4. Publish your page — the widget loads automatically.

Works with WordPress, Squarespace, Wix, Ghost, Hugo, Next.js, and any platform that supports custom HTML. For WordPress, use a "Custom HTML" block in the editor.

Frequently Asked Questions

Are the widgets free?

Yes, 100% free. We only ask that you keep the "Powered by RunBikeCalc" attribution link visible.

Will this slow down my website?

No. Each widget is under 5KB of vanilla JavaScript with zero external dependencies. They load asynchronously and won't block your page rendering.

Will the widget conflict with my CSS?

No. Each widget uses Shadow DOM, which fully encapsulates its styles. Your site's CSS cannot leak in, and the widget's CSS cannot leak out.

Can I use multiple widgets on the same page?

Yes. Each widget creates a unique instance, so you can embed both the pace calculator and heart rate zone calculator on the same page without conflicts.

Do you track my visitors?

No. The widgets contain no tracking code, cookies, or analytics. They run entirely in the browser with no server calls.
