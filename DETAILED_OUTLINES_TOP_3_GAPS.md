## NUMBER 3: RESTING METABOLIC RATE (RMR) CALCULATOR

### Calculator Specifications
- Tool Name: "Resting Metabolic Rate (RMR) Calculator"
- Target Keyword: "resting metabolic rate calculator" (5K+ searches/month)
- Build Time: 3-4 hours
- Expected Monthly Traffic: 100-200 visits

### User Inputs
- Gender: Male / Female
- Age: numeric input (15-100 years)
- Height: with metric/imperial toggle
- Weight: with metric/imperial toggle
- Activity Level: dropdown selection
  - Sedentary
  - Lightly active
  - Moderately active
  - Very active
  - Athlete

### Calculation Methods
- Mifflin-St Jeor formula (recommended, most accurate)
- Harris-Benedict formula (historical comparison)
- Output includes RMR, TDEE, weekly/monthly calorie burn

### Output Display
- Resting metabolic rate in calories/day
- Daily calorie needs by activity level
- Weight loss planning estimates (0.5 lb/week, 1 lb/week)
- Weight maintenance and gain scenarios

### Features
- Unit conversion toggle (metric/imperial)
- Live calculation updates
- Share results capability
- Download as PDF option
- Comparison to average for user's demographic

### Supporting Blog Post
- Title: "Resting Metabolic Rate Calculator: Why Endurance Athletes Need to Know Their RMR"
- Word Count: 1,500-2,000 words
- Sections:
  1. What is RMR and why it matters
  2. How RMR differs for athletes
  3. RMR vs BMR vs TDEE explanation
  4. Using RMR for weight loss planning
  5. How training affects RMR
  6. FAQ about RMR
  7. Integration with nutrition strategy
