Space Requirements

Indoor Space (Minimum)

  • • SkiErg: 2' x 2' floor + 8' ceiling (wall-mounted)
  • • RowErg: 9' x 4' when in use
  • • Wall Balls: 4' x 4' + 10' ceiling minimum
  • • Functional Training: 8' x 8' clear space

Recommended: 12' x 12' minimum (144 sq ft)

Outdoor Space (Sled Work)

  • • Sled Push/Pull: 50m+ straight path
  • • Surface: Grass, turf, or smooth concrete
  • • Farmers Carry: 200m loop or out-and-back
  • • Lunges: 100m path
