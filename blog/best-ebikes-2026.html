Frequently Asked Questions

Class 1 vs Class 2 vs Class 3 e-bikes?

Class 1 — pedal-assist only, 20mph max. Class 2 — throttle + pedal-assist, 20mph max. Class 3 — pedal-assist only, 28mph max (commuter standard). For most buyers, Class 3 commuters or Class 1 for trails/gravel.

How far do e-bikes actually go on one charge?

Depends on motor size, battery capacity, and assist level. Typical ranges: 30-80 miles. Eco mode doubles range. Heavy use (full turbo, hills) halves it. Most commuter e-bikes get 40-60 miles mixed use. Always check real-world reviews, not manufacturer claims.

Do I need insurance for an e-bike?

Most homeowner's/renter's insurance covers e-bikes up to coverage limits (~$1500-3000). For pricier e-bikes, specialized bike insurance (Velosurance, Markel) runs $100-400/year. Check liability coverage too if you commute in traffic.

Bosch vs Shimano vs Specialized motors?

Bosch — smoothest power delivery, most reliable, widest dealer network. Shimano — strong torque, best integration with road bikes. Specialized SL — lightest/quietest, best for road/gravel. All three are excellent; Bosch is the safe default for commute e-bikes.
